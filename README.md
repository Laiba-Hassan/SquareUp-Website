# SquareUp — React + Tailwind

## Setup

```bash
npm install
npm run dev
```

Then open the printed local URL. Build for production with `npm run build`.

## Structure

- `src/pages/` — Home, Services, Process, About, Careers, Contact, Work
- `src/components/` — Navbar, Footer, PageHero, CtaBanner, Faq, ContactForm, Logo, SectionLabel
- Routing via `react-router-dom`
- Styling via Tailwind CSS (dark theme, lime-green accent, matching the provided mockups)
